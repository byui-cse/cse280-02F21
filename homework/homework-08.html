# sample solution
def factor(n):
    for x in range(2, n):
        if n % x == 0: # x evenly divides n
            return (x, n//x)
    return ("Prime")

from math import floor, sqrt
def factor_2(n):
    for x in range(2, floor(sqrt(n)) + 1):
        if n % x == 0: # x evenly divides n
            return (x, n//x)
    return ("Prime")

# Euclid's GCD Algorithm, iterative version

# Method #1
def igcd(x, y):
    while (x > 0):
        r = y%x
        y = x
        x = r
    return y

# Another way, shortcutting the r
def igcd(x, y):
    while x>0:
        (x,y) = (y%x, x)
    return y

print(igcd(0, 1) == 1

Sample Solution 8.6¶

  1. gcd(5,11)

    First find the gcd.

    1. $11 = 5(2) + 1$ <-- gcd

    2. $5 = 1(5) + 0 $

    Now use substitution to find a linear combination of the form $1=s*5 + t*11$.

    1. $1 = 11 - 5(2)$ from step 1

    2. $1 = -2*5 + 11$ rearrange and done

    $s=-2$ and $t=1$

  1. gcd(27,39)

    1. $39 = 27(1) + 12$

    2. $27 = 12(2) + 3$ <-- gcd

    3. $12 = 3(4) + 0$

    Now use substitution to find a linear combination of the form $3=s*27 + t*39$

    1. $3 = 27 - 2(12)$ from step 2

    2. $12 = 39 - 27$ from step 1

    3. $3 = 27 - 2(39 - 27)$ substitute step 5

    4. $3 = 27 - 2(39) + 2(27)$ simplify

    5. $3 = 3(27) - 2(39)$ simplify and rearrange and done

    $s=3$ and $t=-2$
